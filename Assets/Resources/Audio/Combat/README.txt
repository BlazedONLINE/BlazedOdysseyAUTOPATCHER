Combat Audio Resources
====================

This folder contains sound effects for combat system.

Folders:
- Swings/ - Sword swing/whoosh sounds (played when attacking)
- Hits/ - Impact/hit sounds (played when attack connects with target)

Recommended Sound Sources (Free/CC0):
- OpenGameArt.org "Swishes Sound Pack" (CC0): https://opengameart.org/content/swishes-sound-pack
- Pixabay Sword Sounds (Royalty-Free): https://pixabay.com/sound-effects/search/sword/
- Mixkit Sword Effects (Royalty-Free): https://mixkit.co/free-sound-effects/sword/

Instructions:
1. Download sound files from the sources above
2. Place swing/whoosh sounds in Swings/ folder
3. Place hit/impact sounds in Hits/ folder
4. The CombatSoundManager will automatically load them
5. Use the in-game sound selector to test different sounds

Supported formats: .wav, .mp3, .ogg
